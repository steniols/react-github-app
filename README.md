# React Github App

Github application study with express, react and sqlite

## Requirements

1 - Postgress installed
2 - Github APP for Oauth
3 - Node and Npm installed

## Install API

```bash

cp .env.sample .env # and check/update configs
npm install
```

## Start API

```bash
npm start
```

## Client

```bash
cd client
cp .env.sample .env # and check/update configs
npm install
```

## Start Client

```bash
npm start
```
